Star trails

Intellij:
Open project 
Define SDK as ver. 16
Settings -> Invalidate caches (invalidate & restart)
Run main

Eclipse:
Open Project 
Run main

Make sure core.jar is known as a library in the IDE.

Controls:
Arrow keys:
'◄','►' decrease, increase speed
'▼','▲' decrease, increase star trail length

Top left for fps, speed & fade time.
